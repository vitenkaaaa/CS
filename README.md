Проект предназначен для загрузки, скачивания, удаления и поиска файлов в облаке. Включает серверную часть для обработки запросов клиентов и клиентское приложение для взаимодействия пользователей с системой.

Основной функционал:
Загрузка файлов: Пользователи могут загружать файлы в облако.
Скачивание файлов: Возможность скачивать ранее загруженные файлы.
Удаление файлов: Удаление файлов из облачного хранилища.
Поиск файлов: Поиск по имени или другим атрибутам.
Регистрация и аутентификация: Управление доступом пользователей.
Обработка клиентских запросов: Обеспечивает работу с несколькими клиентами одновременно.
Структура проекта:
Server: Серверная часть, обработчик сетевых запросов, управление файлами.
Client: Клиентское приложение для взаимодействия пользователей.
Дополнительные классы: Модели данных, утилиты для работы с файлами и сетью.
Инструкция по сборке и запуску (на базе Maven или без):
Требования:
Java Development Kit (JDK) версии 8 или выше.
Среда разработки (например, IntelliJ IDEA, Eclipse) или командная строка.
Как собрать проект:
Если проект использует Maven:

Склонируйте репозиторий:
git clone https://github.com/vitenkaaaa/CS.git

Перейдите в директорию проекта:
cd CS/CloudStorage

Соберите проект с помощью Maven:
mvn clean install

Запуск сервера:
java -cp target/your-artifact.jar com.yourpackage.Server

Запуск клиента:
Аналогично, запустите класс Client:
java -cp target/your-artifact.jar com.yourpackage.Client
или через IDE.

Дополнительные настройки:
Для работы сервера может понадобиться настроить порт и IP-адрес.
Конфигурационные параметры обычно задаются в файлах настроек или через аргументы командной строки.
